Tags HTML

Tags tipo Block

  • head: Cabeçalho
  • body: Corpo
  • h1..h6: Título
  • p: Parágrafo
  • ol/ul: Lista
  • table: Tabela
  • div: Agrupamento em Block(para formatação/script)
  • etc...

Tags tipo Inline

  • img: Imagem
  • em: ênfase (itálico por default)
  • strong: importância (negrito por default)
  • mark: relevância (destacado por default)
  • ins: inserção (sublinhado por default)
  • del: remoção (sobretachado por default)
  • a href: hyperlink
  • span: Agrupamento Inline(para formatação/script)
  • etc...

Tag <A HREF (Hyperlink)>

Hyperlink interno

Usado tanto para endereço de arquivo quanto de página que está contido dentro do próprio site.

Syntaxe: <a href="endereço relativo ou absoluto interno">Nome explícito do link na página</a>

Hyperlink externo

Usado para endereço de arquivo quanto de página que está contido fora do próprio site. O endereço deve estar completo.

Syntaxe: <a href="endereço completo web">Nome explícito do link na página</a>

Pode também ser usado para endereço de e-mail:

Syntaxe: <a href="mailto:email@gmail.com">Nome explícito do link para e-mail na página</a>

Endereçamento do hyperlink

Adicionando o atributo target="..." podemos definir se o link abre na mesma aba ou em uma nova:

Syntaxe: <a href="link" target="_self">Mesma aba</a>

Syntaxe: <a href="link" target="_blank">Nova aba</a>

Hyperlink para própria página

Atribuindo para qualquer elemento da página um id, podemos posicionar a página diretamente a ele usando o hyperlink:

Syntaxe: <h1 id="primeiro_titulo">Título</h1> <!--Aqui é o local endereçado-->

Syntaxe: <a href="#primeiro_titulo">Pular para primeiro título</a> <!--Referenciando o local Usa-se # antes da chamada do id no Hyperlink-->

Tag Table

Para construir uma tabela, a tag Table usa vários componentes:

Tags usadas para construção de tabelas
TagNívelDescrição
<table></table>0Define o block da table
<caption></caption>1Dentro da table, declara sua legenda
<thead></thead>1Define dentro do block da table, a parte do cabeçalho
<tbody></tbody>1Define dentro do block da table, a parte do corpo
<tfoot></tfoot>1Define dentro do block da table, a parte do rodapé
<tr></tr>2Declara uma linha da tabela
<td></td>3Define uma célula da tabela
<th></th>3Define uma célula de cabeçalho da tabela

Rules CSS

No arquivo CSS declaramos o estilo de cada tag usada no HTML. Para declarar para uma tag inteira, basta:

h1{ "Código de estilo" }

Caso queira alterar o estilo de somente um id da tag dentro do código, chama-se diretamente pelo id:

#id_teste{ "Código de estilo"}
